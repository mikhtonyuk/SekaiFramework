SekaiFramework

An OSGi module platform and service bus implementation for C++

Sekai Framework is a software extensibility platform for C++ projects. Framework provides multiple solutions for developing modular extensible applications. It also includes library for cross-language development, allowing you to easily use C++ in combination with scripting languages.

Dependencies

  • boost C++ library (filesystem, unit-testing)
  • zlib
  • tinyXML

Install

Source distribution includes VisualStudio 2008 solution, and a configured workspace for Code::Blocks

See the API Reference for doxygen-generated documentation.
